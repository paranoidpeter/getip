This tiny bash script fetches ip information of your system. It uses Â `curl` to fetch the global IP from `https://ifconfig.me/` and more informations on this ip are fetched from `https://ipinfo.io/`. Also  local network informations will be displayed with the help of `ipcalc`.

## :hammer_and_wrench: Preparation

1. Install dependencies

```bash
  # Arch based systems
  > pacman -S curl jq ipcalc
  # Debian based systems
  > apt-get install curl jq ipcalc
```

## :cd: Installation

```bash
  > git clone https://github.com/paranoidpeter/getip.git
  > cd getip
  > sudo chmod 755 install.sh
  > sudo ./install.sh
```

## :rocket: Usage
#### Command overview:

```
  > getip [options]           Usage in terminal

  > getip -v                  Get current version
  > getip --version
```

## :wastebasket: Uninstall
```bash
  > cd download-path/getip
  > sudo ./install.sh -u
```
